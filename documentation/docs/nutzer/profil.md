# Profil verwalten

Passe dein Profil an und verwalte deine Kontodaten.

## Profil bearbeiten öffnen

1. Gehe zum **Profil**-Tab
2. Klicke auf **Profil bearbeiten**

## Anzeigename ändern

Dein Anzeigename wird bei deinen Kommentaren und Meldungen angezeigt.

1. Gib den neuen Namen im Textfeld ein
2. Klicke auf **Speichern**
3. Der Name wird sofort aktualisiert

## Profilbild

### Profilbild hochladen

1. Klicke auf **Bild ändern**
2. Wähle ein Bild von deinem Gerät
3. Erlaubte Formate: JPG, JPEG, PNG, WebP
4. Das Bild wird automatisch hochgeladen und angezeigt

!!! tip "Gutes Profilbild"
    - Quadratisches Format funktioniert am besten
    - Klares, gut beleuchtetes Foto
    - Dein Gesicht sollte erkennbar sein

### Profilbild löschen

1. Klicke auf **Bild löschen**
2. Bestätige die Aktion im Dialog
3. Das Standardbild (Icon) wird wiederhergestellt

## Passwort ändern

1. Klicke auf **Passwort ändern**
2. Gib dein **aktuelles Passwort** ein
3. Gib das **neue Passwort** ein
4. Bestätige das neue Passwort
5. Klicke auf **Ändern**

!!! info "Passwort-Anforderungen"
    - Mindestens 8 Zeichen
    - Mindestens eine Ziffer (0-9)
    - Mindestens ein Sonderzeichen (!@#$%^&* etc.)

## Konto löschen

!!! warning "Achtung"
    Das Löschen deines Kontos ist **dauerhaft** und kann nicht rückgängig gemacht werden!

1. Klicke auf **Konto löschen**
2. Bestätige die Warnung im Dialog
3. Dein Konto und alle zugehörigen Daten werden gelöscht
4. Du wirst zur Login-Seite weitergeleitet

Bei der Kontolöschung werden entfernt:

- Dein Benutzerprofil
- Dein Profilbild
- Deine erstellten Meldungen
- Deine Favoriten
- Deine gespeicherten Suchaufträge
- Deine Kommentare
