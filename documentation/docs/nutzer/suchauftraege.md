# Gespeicherte Suchaufträge

Speichere deine Suchfilter, um sie später mit einem Klick wiederzuverwenden.

## Was sind Suchaufträge?

Ein Suchauftrag speichert deine aktuellen Filter:

- Suchbegriff
- Kategorie (Vermisst / Fundtier / Wiedervereint)
- Tierart (Hund / Katze / Kleintier)
- Rasse
- Geschlecht
- Farben

So musst du die Filter nicht jedes Mal neu einstellen.

## Suchauftrag speichern

1. Stelle deine gewünschten Filter auf der **Startseite** ein
2. Klicke auf den **Lesezeichen**-Button (neben "Suchen")
3. Ein Dialog öffnet sich
4. Gib einen **Namen** für den Suchauftrag ein  
   z.B. "Vermisste Katzen schwarz"
5. Klicke auf **Speichern**

!!! tip "Aussagekräftige Namen"
    Wähle einen Namen, der die Filter beschreibt:
    
    - "Vermisste Hunde Hamburg"
    - "Fundkatzen grau getigert"
    - "Wiedervereinte Kleintiere"

## Suchauftrag anwenden

1. Gehe zum **Profil**-Tab
2. Klicke auf **Gespeicherte Suchaufträge**
3. Du siehst eine Liste aller gespeicherten Aufträge
4. Klicke auf einen Suchauftrag
5. Du wirst zur Startseite weitergeleitet
6. Die Filter werden automatisch angewendet

## Suchauftrag löschen

1. Öffne **Profil** → **Gespeicherte Suchaufträge**
2. Finde den Suchauftrag, den du löschen möchtest
3. Klicke auf das **Löschen**-Icon (Papierkorb)
4. Der Suchauftrag wird entfernt

## Anwendungsfälle

### Regelmäßige Suche

Du hast ein Tier verloren und möchtest regelmäßig nach neuen Fundmeldungen suchen:

1. Stelle Filter ein: Tierart, Farbe, Region
2. Speichere als "Meine Suche nach [Tiername]"
3. Wende den Suchauftrag täglich an

### Verschiedene Filtersets

Du hilfst bei der Suche nach mehreren Tieren:

1. Erstelle für jedes Tier einen eigenen Suchauftrag
2. Wechsle schnell zwischen den Aufträgen
3. Behalte den Überblick

### Helfer im Tierheim

Du arbeitest oder hilfst ehrenamtlich:

1. Speichere "Fundtiere heute" (nur Fundtiere)
2. Speichere "Vermisste Hunde" (nur vermisste Hunde)
3. Wende je nach Bedarf den passenden Auftrag an
