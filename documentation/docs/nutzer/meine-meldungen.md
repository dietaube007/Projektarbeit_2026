# Meine Meldungen

Verwalte deine eigenen Meldungen – bearbeite oder lösche sie bei Bedarf.

## Übersicht öffnen

1. Gehe zum **Profil**-Tab
2. Klicke auf **Meine Meldungen**
3. Du siehst eine Liste aller deiner erstellten Meldungen

## Meldungskarten

Jede Meldung zeigt:

- **Bild** (falls vorhanden)
- **Status-Badge** (Vermisst / Fundtier / Wiedervereint)
- **Tierart und Rasse**
- **Standort**
- **Erstellungsdatum**
- **Aktionen** (Bearbeiten, Löschen)

## Meldung bearbeiten

Du kannst deine Meldungen jederzeit aktualisieren, z.B. wenn du neue Informationen hast.

1. Klicke auf das **Bearbeiten**-Icon (Stift)
2. Ein Dialog öffnet sich mit allen Daten
3. Ändere die gewünschten Felder:
   - Status (Vermisst → Wiedervereint)
   - Beschreibung
   - Standort
   - Kontaktdaten
4. Klicke auf **Speichern**
5. Die Änderungen werden übernommen

### Status aktualisieren

Wenn dein Tier gefunden wurde, aktualisiere den Status:

1. Bearbeite die Meldung
2. Ändere den Status auf **Wiedervereint**
3. Speichere die Änderung

!!! success "Gute Nachricht teilen"
    Wenn dein Tier gefunden wurde, setze den Status auf **Wiedervereint**. Das hilft anderen Nutzern zu sehen, dass die Suche erfolgreich war.

## Meldung löschen

!!! warning "Achtung"
    Das Löschen einer Meldung ist **dauerhaft** und kann nicht rückgängig gemacht werden!

1. Klicke auf das **Löschen**-Icon (Papierkorb)
2. Bestätige die Warnung im Dialog
3. Die Meldung wird entfernt

Bei der Löschung werden auch entfernt:

- Das hochgeladene Bild
- Alle Kommentare zur Meldung
- Favorisierungen durch andere Nutzer

## Tipps

- Halte deine Meldungen **aktuell** – füge neue Infos hinzu
- Setze den Status auf **Wiedervereint**, wenn das Tier gefunden wurde
- **Lösche** Meldungen nur, wenn sie nicht mehr relevant sind
