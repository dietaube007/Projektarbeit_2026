# Meldung erstellen

So erstellst du eine Meldung für ein vermisstes oder gefundenes Tier.

## Voraussetzungen

- Du musst angemeldet sein
- Tippe auf den **Melden**-Tab

## Schritt-für-Schritt

### 1. Meldungsart wählen

Wähle zwischen:

- **Vermisst** – Dein Tier ist verschwunden
- **Fundtier** – Du hast ein fremdes Tier gefunden

### 2. Foto hochladen

1. Tippe auf den Foto-Bereich
2. Wähle ein Bild von deinem Gerät
3. Das Bild wird als Vorschau angezeigt

!!! tip "Gutes Foto"
    - Zeige das Tier deutlich
    - Gute Beleuchtung
    - Besondere Merkmale sichtbar

### 3. Tierdaten eingeben

| Feld | Beschreibung | Pflicht |
|------|--------------|---------|
| **Name** | Name des Tieres oder Titel | ✅ |
| **Tierart** | Hund, Katze, Kleintier | ✅ |
| **Rasse** | Spezifische Rasse | Optional |
| **Geschlecht** | Männlich, Weiblich, Unbekannt | Optional |
| **Farben** | Eine oder mehrere Farben | ✅ |

### 4. Beschreibung

Gib eine detaillierte Beschreibung ein:

- Besondere Merkmale (Narben, Flecken, etc.)
- Verhalten
- Halsband oder Chip vorhanden?
- Zusätzliche Informationen

### 5. Standort & Datum

- **Standort**: Wo wurde das Tier zuletzt gesehen / gefunden?
- **Datum**: Wann ist es passiert? (Format: TT.MM.JJJJ)

### 6. Absenden

1. Prüfe alle Angaben
2. Klicke auf **Meldung erstellen**
3. Bei Erfolg erscheint eine Bestätigung
4. Du wirst zur Startseite weitergeleitet

## Meine Meldungen

Deine erstellten Meldungen findest du unter:

**Profil** → **Meine Meldungen**

Dort kannst du:

- Alle deine Meldungen einsehen
- Meldungen bearbeiten (z.B. Status auf "Wiedervereint" setzen)
- Meldungen löschen

Mehr dazu: [Meine Meldungen verwalten](meine-meldungen.md)
