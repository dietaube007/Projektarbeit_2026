# Favoriten

Speichere interessante Meldungen, um sie schnell wiederzufinden.

## Meldung favorisieren

Du kannst jede Meldung zu deinen Favoriten hinzufügen.

### Auf der Startseite

1. Finde eine Meldung, die dich interessiert
2. Klicke auf das **Herz**-Icon
3. Das Icon wird rot – die Meldung ist favorisiert

### Im Detail-Dialog

1. Klicke auf eine Meldung, um Details zu öffnen
2. Klicke auf das **Herz**-Icon
3. Die Meldung wird zu deinen Favoriten hinzugefügt

!!! note "Anmeldung erforderlich"
    Du musst angemeldet sein, um Meldungen zu favorisieren.

## Favoriten anzeigen

1. Gehe zum **Profil**-Tab
2. Klicke auf **Favorisierte Meldungen**
3. Du siehst alle deine gespeicherten Meldungen

## Favorit entfernen

### Aus der Favoritenliste

1. Öffne **Profil** → **Favorisierte Meldungen**
2. Finde die Meldung, die du entfernen möchtest
3. Klicke auf das **X**-Icon oder das rote Herz
4. Die Meldung wird aus deinen Favoriten entfernt

### Auf der Startseite

1. Finde die favorisierte Meldung (rotes Herz)
2. Klicke erneut auf das Herz-Icon
3. Das Icon wird grau – der Favorit wurde entfernt

## Wofür Favoriten nutzen?

- **Vermisste Tiere beobachten**: Speichere Meldungen von Tieren, die du vielleicht gesehen hast
- **Vergleichen**: Speichere ähnliche Fundtiere zum Vergleich
- **Wiederfinden**: Schneller Zugriff auf wichtige Meldungen
- **Kontakt später aufnehmen**: Speichere Meldungen, um später Kontakt aufzunehmen
