# Konto erstellen & Anmelden

## Registrierung

1. Tippe auf **Melden** oder **Profil** in der Navigation
2. Auf der Login-Seite klicke auf **Registrieren**
3. Fülle das Formular aus:
   - **E-Mail**: Deine E-Mail-Adresse
   - **Passwort**: Mindestens 8 Zeichen, eine Ziffer und ein Sonderzeichen
   - **Anzeigename**: Dein Name in der App
4. Klicke auf **Registrieren**
5. Prüfe dein E-Mail-Postfach und bestätige deine Adresse

!!! info "Passwort-Anforderungen"
    - Mindestens 8 Zeichen
    - Mindestens eine Ziffer (0-9)
    - Mindestens ein Sonderzeichen (!@#$%^&* etc.)

## Anmelden

1. Gib deine **E-Mail** und dein **Passwort** ein
2. Klicke auf **Einloggen**
3. Du wirst zur Startseite weitergeleitet

## Ohne Account fortfahren

Wenn du nur stöbern möchtest:

1. Klicke auf **Ohne Account fortsetzen**
2. Du gelangst zur Startseite
3. Melden und Profil bleiben gesperrt

## Abmelden

1. Gehe zum **Profil**-Tab
2. Scrolle nach unten
3. Tippe auf **Abmelden**
4. Du wirst zur Login-Seite weitergeleitet

## Passwort vergessen

Falls du dein Passwort vergessen hast:

1. Gehe zur **Login-Seite**
2. Klicke auf **Passwort vergessen?**
3. Gib deine **E-Mail-Adresse** ein
4. Klicke auf **Link senden**
5. Prüfe dein E-Mail-Postfach
6. Klicke auf den Link in der E-Mail
7. Setze ein **neues Passwort**

!!! info "Keine E-Mail erhalten?"
    - Prüfe deinen Spam-Ordner
    - Stelle sicher, dass die E-Mail-Adresse korrekt ist
    - Warte einige Minuten und versuche es erneut
